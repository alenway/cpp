Java – Data Structures
1. Linear Data Structures (Java)
| Category              | Data Structure        | Java Class / Interface              |
| --------------------- | --------------------- | ----------------------------------- |
| Array                 | Static Array          | `int[]`, `String[]`                 |
| Dynamic Array         | Resizable array       | `ArrayList`                         |
| Linked List           | Doubly Linked List    | `LinkedList`                        |
| Stack                 | LIFO structure        | `Stack`, `Deque`                    |
| Queue                 | FIFO structure        | `Queue`, `ArrayDeque`, `LinkedList` |
| Deque                 | Double-ended queue    | `ArrayDeque`, `LinkedList`          |
| String-based sequence | Character sequence    | `StringBuilder`, `StringBuffer`     |
| Priority-based linear | Binary heap-backed    | `PriorityQueue`                     |
| Special Lists         | Synchronized list     | `Vector`                            |
| Streams               | Sequential processing | `Stream<T>`                         |



2. Non-Linear Data Structures (Java)
| Category           | Data Structure              | Java Class / Interface                  |
| ------------------ | --------------------------- | --------------------------------------- |
| Tree               | General Tree (not built-in) | Implemented manually                    |
| Binary Tree        | Custom BT                   | Implement manually                      |
| Binary Search Tree | Custom BST                  | Implement manually                      |
| Balanced Trees     | Red-Black Tree              | `TreeMap`, `TreeSet`                    |
| Heap               | Min/Max Heap                | `PriorityQueue`, custom heap            |
| Hash Table         | Hash Map                    | `HashMap`, `LinkedHashMap`, `Hashtable` |
| Set Structures     | Hash Set                    | `HashSet`                               |
| Ordered Set        | LinkedHashSet               | `LinkedHashSet`                         |
| Graph              | Adjacency list/matrix       | Custom (no built-in `Graph`)            |
| Trie               | Prefix Tree                 | Custom implementation                   |
| Segment Tree       | Range queries               | Custom implementation                   |
| Fenwick Tree       | BIT                         | Custom implementation                   |


C++ – Data Structures
1. Linear Data Structures (C++)
| Category        | Data Structure     | C++ Type / STL                     |
| --------------- | ------------------ | ---------------------------------- |
| Array           | Static array       | `int arr[]`                        |
| Array           | Fixed-size array   | `std::array`                       |
| Dynamic Array   | Vector             | `std::vector`                      |
| Linked List     | Singly Linked List | `std::forward_list`                |
| Linked List     | Doubly Linked List | `std::list`                        |
| Stack           | LIFO structure     | `std::stack`                       |
| Queue           | FIFO structure     | `std::queue`                       |
| Priority Queue  | Heap-based         | `std::priority_queue`              |
| Deque           | Double-ended queue | `std::deque`                       |
| String sequence | Mutable/Immut      | `std::string`, `std::stringstream` |

2. Non-Linear Data Structures (C++)
| Category              | Data Structure        | C++ Type / STL                          |
| --------------------- | --------------------- | --------------------------------------- |
| Tree                  | General Tree          | Custom                                  |
| Binary Tree           | BT                    | Custom                                  |
| Binary Search Tree    | BST                   | Custom                                  |
| Balanced Trees        | Red-Black Tree        | `std::map`, `std::set`                  |
| Unordered Hash Tables | Hash Map              | `std::unordered_map`                    |
| Set (Hash)            | HashSet               | `std::unordered_set`                    |
| Graph                 | Adjacency list/matrix | Custom or using STL containers          |
| Heap                  | Min/Max Heap          | `std::priority_queue`, `std::make_heap` |
| Trie                  | Prefix tree           | Custom                                  |
| Segment Tree          | Range queries         | Custom                                  |
| Fenwick Tree          | BIT                   | Custom                                  |
